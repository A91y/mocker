# Music

## Module to generate music related entries.

:::info

For more **schema options** refer, [faker.js documentation](https://fakerjs.dev/).

:::

### genre

Returns a random music genre.

#### output

```json
Soul
```

### songName

Returns a random song name.

#### output

```json
Somethin' Stupid
```
