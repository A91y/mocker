# Color

## Module to generate colors.

:::info

For more **schema options** refer, [faker.js documentation](https://fakerjs.dev/).

:::

### hsl

Returns an HSL color.

#### output

```json
[198, 0.59, 0.72]
```

### humanColor

Returns a random human readable color name.

#### output

```json
red
```

### rgb

Returns an **RGB** color.

#### output

```json
0xffffff
```
