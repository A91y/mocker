# Science

## Module to generate science related entries.

:::info

For more **schema options** refer, [faker.js documentation](https://fakerjs.dev/).

:::

### Chemical unit

Returns a random periodic table element.

#### output

```json
{ "symbol": "Tb", "name": "Terbium", "atomicNumber": 65 }
```
